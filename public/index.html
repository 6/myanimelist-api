MyAnimeList Unofficial API

This is the unofficial API for MyAnimeList, the best site there is to keep track of the anime you've watched or plan to watch. It's like Facebook for anime lovers.

This API is a supplement to the official API that's currently being developed and is not ready for prime-time use yet.

For announcements and updates, follow @chuyeow on Twitter.
